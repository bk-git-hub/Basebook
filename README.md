# Basebook

Basebook is a Sweetbook Book Print API powered web service for baseball fans to record game experiences and turn a season of memories into a printed book.

## Workspace Structure

- `apps/web` — Next.js frontend
- `apps/api` — Nest.js backend
- `packages/contracts` — shared types
- `data` — demo datasets
- `docs` — planning, reference, and presentation docs

## Status

Initial monorepo scaffold is ready. Product features and API integration will be added next.
