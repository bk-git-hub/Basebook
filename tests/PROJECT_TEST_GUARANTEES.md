# Project Test Guarantees

현재 자동 테스트가 보장하는 범위입니다.

## 보장하는 것

- API 헬스체크가 응답한다.
- 경기 후보 조회가 필터링과 validation을 수행한다.
- 경기 일지 목록, 상세, 생성, 수정, 삭제 API가 동작한다.
- 로컬 이미지 업로드 API가 파일을 저장하고 다시 제공한다.
- 웹 홈, 소개, 시즌, 기록 상세, 기록 수정 화면이 렌더링된다.
- 브라우저에서 기록 생성, 수정, 삭제 흐름이 API와 연결된다.

## 보장하지 않는 것

- 로그인과 사용자별 권한
- 외부 커머스 연동
- 운영 환경 데이터 마이그레이션
